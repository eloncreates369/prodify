# Prodify

Prodify is a productivity dashboard that helps users manage tasks, habits, timers, reminders, and productivity analytics.

## Features

- User authentication (JWT)
- Task management
- Habit tracking
- Timer tracking
- Productivity analytics dashboard
- MongoDB cloud database
- Responsive UI

## Tech Stack

Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

Backend
- Node.js
- Express
- MongoDB
- JWT Authentication
